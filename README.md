# Grocery Store Management System (DSA - C++)

## 📌 Description
This project is a Grocery Store Management System developed in C++ using Data Structures and Algorithms (DSA).  
It simulates a basic store system where products can be managed efficiently.

## 🚀 Features
- Add new products
- Display product list
- Search products
- Update product details
- Delete products
- Billing system

## 🧠 DSA Concepts Used
- Arrays / Vectors
- Structures (struct)
- Searching Algorithms
- Sorting Techniques
- File Handling (optional)

## 🛠 Technologies Used
- C++
- DSA Concepts

## ▶️ How to Run
1. Open the project in any C++ compiler (Dev C++, CodeBlocks, VS Code)
2. Compile the main.cpp file
3. Run the program
